# Создание программы «Todo list»

## Versions

**1.0.0** - Todo list на основе объекта

**Интерфейс**: консоль

**Описание:** Одна задача - свойство в объекте. Ключ - название задачи. Значение - статус задачи.

**Функционал**:

- Добавление задачи
- Удаление задачи
- Изменение статуса задачи
- Вывод списка задач в консоль
